1. Professor tutorial

STEP 1.

① ID 또는 E-mail을 입력합니다.
② 비밀번호를 입력합니다.
③ 로그인 버튼을 클릭하면 로그인 할 수 있습니다.

STEP 2.

① 강좌등록을 등록할 수 있는 버튼입니다.
② 강좌삭제를 할 수 있는 버튼입니다.
③ 과목명을 클릭하면 특정 과목 페이지로 이동합니다.

STEP 2.1 강좌등록

① 강좌등록 버튼을 통해 강좌를 개설합니다.

② 강의 명을 입력합니다.
③ 강좌 개설 버튼을 누르면 강좌가 개설됩니다.

STEP 2.2 강좌삭제

②강좌삭제 수정 버튼을 통해 강좌를 삭제합니다.

①삭제 버튼을 누르면 해당 강좌가 삭제됩니다.

STEP 2.3 강좌 시작하기

① 강의를 추가할 수 있는 버튼입니다.
② 강의를 삭제할 수 있는 버튼입니다.
③ 슬라이드 쇼를 할 수 있는 버튼입니다.
④ 마인드맵을 볼 수 있는 버튼입니다.

STEP 2.3.1 강좌 추가하기

① 강의추가 버튼을 누릅니다.
-> 강의명을 쓰고 파일추가 버튼을 누릅니다.
-> 추가할 파일을 선택 후 열기 버튼을 누릅니다.
-> 강의 추가하기 버튼을 누르면 강의가 업로드 됩니다.

STEP 2.3.2

① 강의를 수정할 수 있는 버튼입니다.
② 강의를 삭제할 수 있는 버튼입니다.

STEP 2.3.2.1 강좌 수정하기

-> 강의 수정을 클릭합니다.
-> 수정될 파일을 선택 후 열기 버튼을 클릭합니다.
-> 강의가 수정됩니다.

STEP 2.3.2.2 강좌 삭제하기

삭제하고자 하는 강의의 삭제 버튼을 클릭합니다.
-> 강의가 삭제됩니다.

STEP 2.3.3

STEP 2.3.4

STEP 3.이해도 평가하기

① 설문 내용을 작성합니다.
② 작성한 내용을 학생에게 송신합니다.

교수는 학생들의 강의 이해도를 확인할 수 있습니다.

STEP 4.강의자료 -> 마인드맵.

STEP 4.1

강의 목차 있는 강의자료 -> 마인드맵 생성

동영상 삽입

STEP 4.2

파일 -> 열기 버튼 클릭

강의자료 선택 -> 열기 버튼 클릭

① 강의자료 목차 자동생성 ② 초기 마인드맵 생성 -> 마인드맵 최상단 노드 입력.

② 마인드맵 현재 노드 선택(마우스 클릭 또는 키보드 방향키 입력, 선택된 노드는 회색) -> ① 하위노드에 추가될 강의자료 페이지 더블클릭 -> ②최상단 노드의 하위노드 생성.

마인드맵 완성 -> ID 입력 -> Password 입력 -> Upload 버튼 클릭 -> 마인드맵 Upload 완료.

강의 목차가 없는 강의 자료 -> 마인드맵 동영상 Tutorial.

2. Student tutorial

STEP 1.

① ID 또는 E-mail을 입력합니다.
② 비밀번호를 입력합니다.
③ 로그인 버튼을 클릭하면 로그인 할 수 있습니다.


STEP 2.

① 자신이 등록한 과목명이 표시됩니다.
    자신이 수강하고 있는 강의의 과목명이 표시됩니다.
    과목명을 클릭하면 해당 강의를 수강할 수 있습니다.
② 강의의 상태를 표시합니다.
    온라인으로 표시된 강의는 현재 강의가 진행중인 상태입니다.
    오프라인으로 표시된 강의는 현재 강의가 진행되지 않고 있는 상태입니다.
    온라인으로 표시된 강의의 과목명을 클릭할 경우
    해당 강의를 수강할 수 있습니다.



STEP 3.

수업이 진행중인 강좌에 들어갔을 경우의 화면입니다.
① 수업이 진행된 경로를 선으로 표시합니다.
지나온 경로는 파란색으로 표시되며
아직 배우지 않은 목차 경로의 선은 검은색으로 표시됩니다.
② 현재 강의가 진행중인 목차입니다.
진행 중인 목차에 파란색으로 원을 둘러 표시됩니다.
③ 현재 배우고 있는 목차 다음에 배울 목차입니다.
다음에 진행될 목차는 파란색 점선으로 표시됩니다.

슬라이드가 진행될 때마다 표시되는 경로와 목차의 표시가 실시간으로 달라집니다.



STEP 4.

① 특정 노드를 클릭하면
② 노드에 해당되는 강의자료 슬라이드를 볼 수 있습니다.


STEP 5.

① 밑줄이나 그림을 그릴 수 있는 펜을 활성화 합니다.
② 필기 했던 것들을 지울 수 있습니다.
③ 메모를 통해 필기를 할 수 있습니다.
④ 메모장을 통해 필기가 가능하고 메모장 윗부분의 접기를 클릭할 경우
⑤ 필기를 접은 상태로 바뀌게 됩니다.
